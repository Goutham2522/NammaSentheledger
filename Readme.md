# Namma Senthe Ledger

A modern Android ledger application developed using Kotlin and Android Studio for managing customers, balances, credits, and payments digitally.

---

## Features

* Add and manage customers
* Store phone numbers with country code picker
* Credit and payment transaction management
* Live balance updates
* Recent transaction dashboard
* Transaction history for every customer
* Delete transactions
* Persistent local storage using SharedPreferences + Gson
* Beautiful splash screen
* Modern UI with Material Design
* Custom bottom navigation
* Scrollable dashboard and transaction history

---

## Technologies Used

* Kotlin
* Android Studio
* XML Layouts
* RecyclerView
* SharedPreferences
* Gson Library
* Material Components
* Country Code Picker Library
